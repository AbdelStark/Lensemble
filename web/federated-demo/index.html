Lensemble Federated Run Demo

Simulated demo. Includes a frontend-only run simulator and a local backend mode for QR joins, WebSocket orchestration, bounded tiny browser update vectors, aggregation, evidence export, and inference. No backend, no real browser training, and no benchmark success is claimed for the simulator slice; the dynamic-env result is an educational systems demo and does not materially beat local-only training.

Tapestry-like real-LeWM mode. The real-lewm-tworooms learner path adapts a real LeWorldModel TwoRooms checkpoint: browsers run checkpoint-backed inference, keep rollouts resident, train a bounded adapter subset, and federate clipped adapter deltas. It is checkpoint adaptation, not full from-scratch LeWM browser pretraining, and not a paper-scale benchmark claim.
